# GA-Blog
## Date: 1/10/2025
### By: Rabab Hasan Isa and Eman Qarooni

***
### Description
Blog website for our class daily life, where you can post a comment, a picture or a video and others users can interact with your posts.
### ***Technologies***
  * CSS
  * node.js
  * HTML
  * Mongoose
  * express
  * ejs
  * Visual Studio Code
